# USE CASE 24: Produce a Report on Regional Population Balance

## CHARACTERISTIC INFORMATION

### Goal in Context
As an Admin, I want a report showing, for each region, the total population, people living in cities, and people not living in cities so that I can compare population distribution across regions.

### Scope
System

### Level
Primary task

### Preconditions
* Admin has access the database system.  
* The world population data is available and already stored in the database.

### Success End Condition
A correctly formatted report displaying the total population data of people living and not living for each region is generated and displayed to the Admin.

### Failed End Condition
No report is produced.

### Primary Actor
Admin

### Trigger
Admin requests a report showing population balance for each region.

## MAIN SUCCESS SCENARIO
1. The Regional Population Balance Report is requested by the Admin.  
2. A complete report showing total population, people living in cities, and people not living in cities for each region is generated.  
3. The generated report is displayed on the screen for the Admin to view.

None
